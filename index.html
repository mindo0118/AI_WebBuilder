About Me

프로젝트: AI 웹페이지 빌더 (Viz-Page AI)

상품 이미지 1장과 키워드만으로 완성형 웹페이지(.zip)를 생성하는 AI 솔루션

문제 정의

  • 소상공인은 전문적인 판매 페이지 제작을 위한 코딩/디자인 지식이 부족합니다.
  • 웹페이지 제작 외주는 비용 부담이 크고, 쇼핑몰 솔루션은 상세페이지 제작에 많은 시간이 필요합니다.

프로젝트 목표

상품 이미지 1장과 키워드 입력만으로, AI가 자동으로 톤앤매너에 맞는 완성형 웹페이지(.zip)를 즉시 제공하여 누구나 전문적인 판매 페이지를 가질 수 있도록 돕습니다.

아키텍처 (Architecture)

듀얼 AI API 아키텍처

기능별로 전문화된 두 종류의 클라우드 AI API를 조합하는 구조를 채택했습니다.

1. AI '두뇌' (Gemini Pro Vision)
Google 멀티모달 LLM. 사용자 이미지/텍스트를 '이해'하고, 판매 문구, HTML/CSS 코드, 배너 이미지 프롬프트를 도출합니다.
2. AI '아티스트' (Stability AI API)
Gemini가 도출한 프롬프트를 받아 고품질 배너 이미지를 '빠르게' 생성하는 클라우드 API 역할을 합니다.
3. 백엔드 (FastAPI)
전체 프로세스를 지휘하는 '컨트롤 타워'. (파일 처리, API 호출, .zip 패키징 및 전송)

핵심 기능

멀티모달 AI를 활용한 HTML/CSS 코드 및 카피라이팅 동시 생성

  • 사용자가 상품 이미지와 키워드를 업로드하면 FastAPI가 Gemini Pro Vision API로 전송합니다.
  • Gemini가 이미지와 키워드를 조합, 분석하여 판매 문구와 웹페이지 레이아웃(HTML/CSS)을 동시에 생성합니다.
  • 웹페이지에 어울릴 배너 이미지의 구체적인 생성 프롬프트까지 생성하여 백엔드로 반환합니다.

클라우드 AI 연동을 통한 고속 배너 이미지 생성 및 최종 웹페이지 패키징

  • 백엔드는 Gemini로부터 받은 프롬프트를 즉시 Stability AI API로 전송하여 배너 이미지를 생성합니다. (사용자 PC 자원 소모 없음)
  • FastAPI 서버는 생성된 index.html, style.css, banner.jpg와 원본 product.jpg를 모두 취합합니다.
  • Python zipfile 라이브러리를 사용해 메모리 상에서 .zip 파일로 압축한 뒤, 사용자에게 즉시 다운로드로 전송합니다.
